## Description

MoneyGrabber REST API for calculating commissions.

## Installation

```bash
# Install dependencies
$ npm install

# Update currency rates cache
$ npm run update:rates
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# Usage:

Access from browser: `http://localhost:3000/` (Swagger UI).

POST to `http://localhost:3000/commission` with external clients.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
